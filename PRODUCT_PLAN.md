# 保险内容创作者情报站 - 产品规划

> 版本：v1.0 | 日期：2026-04-22 | 状态：规划中

**关键决策（2026-04-22）**：
- 产品名：**爆款情报站**
- AI 服务：Claude (Anthropic)
- 团队规模：10人以上（中大型团队）

---

## 一、产品定位

**产品名称**（待定）：
- 灵感库
- 爆款情报站
- 内容雷达
- 创作者工作站

**一句话描述**：
> 保险内容创作者的"信息中台 + 爆款标题工厂"，一个平台搞定热点追踪、行业动态、内容学习和标题生成。

**核心价值**：
1. **信息差变现** - 第一时间捕捉各行业热点，转化为保险内容素材
2. **爆款公式化** - 用 AI 能力拆解爆款规律，批量生成高概率标题
3. **一站式工作流** - 从找选题 → 看案例 → 生成标题，全流程闭环

---

## 二、功能模块

### 2.1 情报中心（首页）

**定位**：信息聚合首页，快速浏览当日重要情报

**功能**：
- 📊 **今日热词榜**：实时更新的热点词汇 TOP 20
- 🔥 **紧急提醒**：重要政策、产品变动等需即时关注的信息
- 📰 **情报速递**：按行业分类的热点信息流
- ⏰ **定时推送**：每日/每周汇总报告（邮件/飞书/微信）

**行业分类**：
| 行业 | 信息来源 | 更新频率 |
|------|----------|----------|
| 保险 | 监管局官网、保险公司公告、险企公众号 | 实时 |
| 金融 | 央行、证监会、基金/券商公告 | 实时 |
| 教育 | 教育部、各地教育局、学校通知 | 日更 |
| 科技/AI | 科技媒体、AI 产品动态、技术博客 | 实时 |
| 社会热点 | 微博热搜、知乎热榜、抖音热榜 | 实时 |

---

### 2.2 行业动态

**定位**：保险行业专属情报

**功能**：
- 📋 **政策速递**：监管政策、法规调整的摘要 + 原文链接
- 🏢 **产品动态**：新产品上架、旧产品调整、责任条款变更
- 📈 **公司新闻**：险企重要动态（上市、融资、高管变动等）
- 💬 **行业观点**：保险公司/代理人发布的专业内容

**信息源**：
- 中国银保监会官网
- 各保险公司官网/公众号
- 保险行业协会
- 保险媒体（今日保、保险观察等）

---

### 2.3 爆款案例库

**定位**：收集整理各平台保险类爆款内容

**功能**：
- 🔍 **多平台搜索**：小红书、公众号、抖音、视频号
- 📊 **数据看板**：点赞、收藏、评论、转发数据
- 🏷️ **标签分类**：按险种（医疗险/重疾险/储蓄险）、按话题、按风格
- 📝 **内容原文**：爆款笔记/文章的完整内容
- 🧠 **爆款分析**：AI 自动分析爆款原因（情绪点、共鸣点、钩子）

**平台对接**：
- 小红书：官方 API / 第三方数据平台
- 公众号：搜狗搜索 / 第三方平台
- 抖音：巨量算数 / 第三方平台

---

### 2.4 标题生成器

**定位**：AI 驱动的爆款标题工厂

**功能**：
- 🎯 **关键词输入**：输入话题关键词（1-3个）
- 🚀 **一键生成**：批量生成 10-20 个标题候选
- 📊 **评分预测**：AI 预测每个标题的爆款概率（1-10分）
- 🏷️ **标签推荐**：自动推荐适合的小红书标签
- 💾 **收藏管理**：收藏好的标题，分类管理

**标题类型支持**：
| 类型 | 示例 | 适用场景 |
|------|------|----------|
| 震惊体 | "99% 的人都不知道的保险真相" | 引起好奇 |
| 数字体 | "每月 200 块，给家人 300 万保障" | 具体化 |
| 故事体 | "我用 3 万保费，换来 50 万理赔" | 真实案例 |
| 对比体 | "有了社保，还需要商业保险吗？" | 答疑解惑 |
| 情绪体 | "看完这篇，我立刻给娃买了保险" | 情绪共鸣 |
| 反差体 | "保险代理人不会告诉你的 5 件事" | 揭露真相 |

---

### 2.5 个人中心

**功能**：
- 👤 **账号管理**：登录/注册/个人设置
- 📁 **收藏夹**：收藏的情报、案例、标题
- 📊 **使用统计**：查看记录、使用习惯分析
- ⚙️ **偏好设置**：关注行业、关键词过滤、通知偏好

### 3.3 数据库设计

#### 用户表（users）
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role VARCHAR(20) DEFAULT 'member', -- admin/member
    created_at TIMESTAMP DEFAULT NOW(),
    last_login TIMESTAMP,
    preferences JSONB DEFAULT '{}' -- 关注行业、关键词等
);
```

#### 情报表（intelligence）
```sql
CREATE TABLE intelligence (
    id SERIAL PRIMARY KEY,
    title VARCHAR(500) NOT NULL,
    summary TEXT,
    content TEXT,
    source VARCHAR(100), -- 来源平台
    source_url VARCHAR(1000),
    category VARCHAR(50), -- insurance/finance/education/tech/social
    tags TEXT[], -- 标签数组
    hot_score INTEGER DEFAULT 0, -- 热度分
    publish_time TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW(),
    INDEX idx_category (category),
    INDEX idx_publish_time (publish_time),
    INDEX idx_hot_score (hot_score)
);
```

#### 爆款案例表（viral_cases）
```sql
CREATE TABLE viral_cases (
    id SERIAL PRIMARY KEY,
    platform VARCHAR(20), -- xhs/wx/douyin
    title VARCHAR(500),
    content TEXT,
    author VARCHAR(100),
    likes_count INTEGER DEFAULT 0,
    favorites_count INTEGER DEFAULT 0,
    comments_count INTEGER DEFAULT 0,
    shares_count INTEGER DEFAULT 0,
    url VARCHAR(1000),
    cover_image VARCHAR(500),
    tags TEXT[],
    insurance_type VARCHAR(50), -- 关联险种
    viral_score FLOAT, -- AI 计算的爆款分
    analysis TEXT, -- AI 爆款分析
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 标题收藏表（saved_titles）
```sql
CREATE TABLE saved_titles (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    keyword VARCHAR(100),
    generated_titles JSONB, -- 批量生成的标题
    final_title VARCHAR(200),
    status VARCHAR(20) DEFAULT 'draft', -- draft/used/archived
    notes TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 采集任务表（scrape_tasks）
```sql
CREATE TABLE scrape_tasks (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    source VARCHAR(100),
    source_url VARCHAR(1000),
    schedule VARCHAR(50), -- cron 表达式
    last_run TIMESTAMP,
    next_run TIMESTAMP,
    status VARCHAR(20) DEFAULT 'active', -- active/paused/error
    created_at TIMESTAMP DEFAULT NOW()
);
```

## 六、AI 能力设计

### 6.1 标题生成 Prompt

```
你是一个专注于小红书保险内容创作的专业标题专家。

用户输入的关键词：{keywords}

请根据以下原则生成 10 个爆款标题：

1. **情感共鸣**：触及用户痛点（健康焦虑、养老焦虑、家庭责任）
2. **数字钩子**：使用具体数字增加可信度
3. **对比反差**：制造认知冲突
4. **疑问引导**：引发好奇心
5. **情绪刺激**：惊讶、恐惧、期待等情绪

每个标题需要包含：
- 标题内容
- 类型（震惊体/数字体/故事体等）
- 爆款概率评分（1-10分）
- 适用场景说明

输出格式：JSON
```

### 6.2 爆款分析 Prompt

```
请分析以下小红书爆款笔记的爆款原因：

标题：{title}
内容：{content}
数据：点赞 {likes} | 收藏 {favorites} | 评论 {comments}

分析维度：
1. 标题吸引力（钩子、情绪、数字等）
2. 内容结构（开头/中间/结尾）
3. 选题角度（为什么能引发讨论）
4. 可复用的元素

输出：
- 爆款因素总结
- 可模仿的写作公式
- 生成类似标题的建议
```
