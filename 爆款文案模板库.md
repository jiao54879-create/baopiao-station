## 一、标题模板（8种类型）

### 1. 震惊体
```
格式：数字 + 极端结果
示例：
- 90%的人都不知道！买保险前一定要看这5点
- 重疾险千万别乱买！看完这3点能省好几万
- 为什么保险越早买越好？看完后悔知道太晚了
```

### 2. 数字体
```
格式：数字 + 具体内容
示例：
- 2024年最值得买的5款重疾险，看完不踩坑
- 家庭保险配置方案：年收入20万怎么买最划算
- 宝宝保险1000元搞定！这套方案建议收藏
```

### 3. 故事体
```
格式：经历/案例 + 结论
示例：
- 从业10年的保险经理，告诉你买保险的3个真相
- 我妈买保险被坑了5万，这些坑你一定要避开
- 邻居的理赔经历让我明白了为什么要买重疾险
```

### 4. 对比体
```
格式：A vs B + 结论
示例：
- 达尔文8号 vs 超级玛丽10号，到底选哪个？
- 医疗险VS重疾险，90%的人都分不清
- 好医保 vs 微医保，2024年最新测评对比
```

### 5. 情绪体
```
格式：情绪词 + 痛点
示例：
- 后悔没早知道！保险这5个坑千万别踩
- 扎心了！买保险最容易犯的10个错误
- 太实用了！原来保险要这样买才划算
```

### 6. 反差别
```
格式：颠覆认知 + 解释
示例：
- 保险经纪人不推荐的保险，你买了吗？
- 为什么我不建议给父母买寿险？
- 孩子最重要的保险，很多家长都忽略了
```

### 7. 疑问体
```
格式：问题 + 答案预告
示例：
- 重疾险到底怎么选？看完这篇你就懂了
- 医疗险和重疾险有什么区别？看完不再纠结
- 增额终身寿险值得买吗？3分钟讲清楚
```

### 8. 实用体
```
格式：方法/攻略 + 受众
示例：
- 2024年保险配置攻略，三口之家必看
- 30岁人群保险配置方案，建议收藏
- 父母保险怎么买？这份指南请收好
```

## 二、内容模板（8种类型）

### 1. 产品测评型
```
结构：
1. 开篇引入（痛点/热点）
2. 产品基本信息

4. 优缺点分析
5. 适合人群
6. 总结推荐

示例开头：
最近很多朋友问我XX产品怎么样，今天来详细测评一下...
```

### 2. 避坑指南型
```
结构：
1. 痛点引入（很多人踩坑）
2. 坑点1 + 案例/数据
3. 坑点2 + 案例/数据
4. 正确做法

```

1. 问题引入
2. 概念解释（通俗易懂）

4. 注意事项

```
结构：

### 5. 人群分析型

### 6. 政策解读型
```
结构：
1. 政策背景
2. 主要变化
3. 对消费者的影响
4. 应对建议

示例开头：
保险新规出台，对我们有什么影响？...
```

### 7. 热点借势型
```
结构：
1. 热点事件引入
2. 与保险的关联
3. 风险分析
4. 保障建议

示例开头：

```

### 8. 年终盘点型
```
结构：
1. 年度回顾
2. 产品榜单
3. 行业趋势
4. 新年建议

示例开头：

## 四、各平台适配

### 微信公众号
- 标题：25字以内，带悬念
- 正文：1500-2500字，分段清晰
- 配图：封面图+文中插图

### 小红书
- 标题：20字以内，带emoji
- 正文：500-800字，重点加粗
- 配图：9宫格或单图精修

### 知乎
- 标题：问句形式
- 正文：专业详尽，可3000字+
- 配图：数据图表更佳

### 微博
- 标题：热搜话题格式 #关键词#
- 正文：500字以内，金句突出
- 配图：一张精修图

## 五、爆款公式

### 标题公式
```
数字 + 痛点 + 解决方案 = 高点击标题
情绪词 + 颠覆认知 + 具体说明 = 传播标题
```

### 内容公式
```
开头：痛点/故事引入（30%）
正文：专业内容+案例（50%）
结尾：总结+行动号召（20%）
```

### 互动公式
```
提问：你有没有遇到过XX情况？
投票：你选择A还是B？
转发：转给家人看看
收藏：建议收藏备用
```
